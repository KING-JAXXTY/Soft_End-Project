# Scholarship Management System

Full-stack scholarship management platform with student, sponsor, and admin portals.

## Live Demo
Deployed on Vercel

## Features
- Student scholarship browsing and applications
- Sponsor scholarship management
- Admin oversight and approval system
- Real-time messaging
- Forum discussions
- Certificate generation

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Deployment: Vercel Serverless
